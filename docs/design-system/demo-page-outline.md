# Design System Demo Page — Outline

Goal: Showcase all core tokens and components in one place for visual QA and iteration.

## Sections
1) **Colors & Gradients**: swatches for bg/surface/hover/primary/accents/danger; hero gradient; chart gradient sample.
2) **Typography**: Plus Jakarta Sans samples (headings, body, captions), weights/sizes.
3) **Buttons**: primary, ghost, destructive, icon-only; hover/active states.
4) **Cards**: default card, hover glow; KPI card with sparkline; data card with divider.
5) **Inputs**: text fields, selects, toggles, checkboxes, date picker; focus rings.
6) **Tabs & Pills**: underline tabs; status badges.
7) **Tables**: dense table example with hover row, sticky header.
8) **Panels/Drawers**: side panel with header/body/footer; scrolling content.
9) **Charts**: line + gradient area with custom tooltip; bar with gradient; pie/radar sample; tooltip and annotation examples using our theme.
10) **Micro-interactions**: hover glow demo, icon micro-animations, cursor-follow glow sample.
11) **3D placeholder**: embed block showing how the viewer container looks in the theme (static placeholder).
12) **Map placeholder**: styled map container with dark theme framing.
